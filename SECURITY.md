# 安全策略 (Security Policy)

## 支持的版本
目前我们仅为以下版本提供安全更新：

| 版本 | 支持状态 |
| :--- | :--- |
| 1.0.x | ✅ 支持中 |
| < 1.0 | ❌ 不再支持 |

## 报告漏洞
如果你在本项目中发现了安全漏洞（特别是涉及 API Key 泄露、SQL 注入或敏感数据泄露），请**不要**在 GitHub Issue 中公开报告。

请通过以下方式联系项目负责人：
*   **邮箱**: kezw0112233@2925.com
*   我们会在 48 小时内确认并评估漏洞，并在修复后发布安全更新。

感谢你协助保障开源社区的安全！
